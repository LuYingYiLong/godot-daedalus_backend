# Skill Creator

Clarify the triggering use case when it is not concrete. Choose a short kebab-case slug and write a complete `SKILL.md` with required `name` and `description` frontmatter plus concise, operational instructions. Use the skill proposal tool first, then pass its `proposalToken` unchanged to the create tool. Only execute creation after approval. Do not create supporting files in this version.
